**RESEARCHER_SYSTEM_PROMPT (export as const string):**
```
You are a content strategist and SEO researcher. Your job is to identify the highest-opportunity content topics for a business to dominate AI-generated search results (ChatGPT, Gemini, Perplexity citations).

Use Google Search to:
1. Find the top 5 competitor sites in the given industry/service area and analyze their content
2. Search for common questions people ask about the industry in the service area
3. Look for what AI tools (ChatGPT, Gemini, Perplexity) answer when asked about services in this industry
4. Identify gaps: high-authority topics that competitors are thin on or missing

Return ONLY a valid JSON array of 5-10 topics. No prose, no markdown code fences, no explanation. Raw JSON array only.

Each topic must use EXACTLY this structure:
[
  {
    "rank": 1,
    "title": "...",
    "targetQueries": ["...", "..."],
    "authorityGapScore": 85,
    "recommendedSchemaTypes": ["Article", "FAQPage"],
    "faqClusters": ["What is...?", "How much does...?"],
    "estimatedCitationLift": "4-6 AI citations/month"
  }
]

Rank topics by authorityGapScore descending (highest gap = rank 1).
```

**buildPrompt function (not exported):**
Takes ClientConfig, returns prompt string. If existingContentUrls has entries, append: "Client's existing content (these are already covered — do NOT recommend them):\n{urls.join('\n')}".

**generateAuthorityMap function (export):**
Signature: `export async function generateAuthorityMap(config: ClientConfig): Promise<AuthorityMapResult>`

Implementation:
1. Instantiate `const google = createGoogleGenerativeAI()` (reads GOOGLE_GENERATIVE_AI_API_KEY from env)
2. Call generateText with:
   - model: google('gemini-2.0-flash')
   - tools: { google_search: google.tools.googleSearch() }
   - stopWhen: stepCountIs(10)
   - system: RESEARCHER_SYSTEM_PROMPT
   - prompt: buildPrompt(config)

   If google.tools.googleSearch() causes a TypeScript type error, use fallback:
   - Remove tools property
   - Add providerOptions: { google: { useSearchGrounding: true } }

3. Strip any markdown code fences from text (regex replace /^```json\n?|```$/gm, '')
4. Parse JSON: `const topics: AuthorityMapTopic[] = JSON.parse(text)`
5. Derive month: `const now = new Date(); const month = \`\${now.getFullYear()}-\${String(now.getMonth() + 1).padStart(2, '0')}\``
6. Return: `{ clientId: config.clientId, month, topics }`

Throw on JSON parse failure with message: `Authority map research returned invalid JSON. Raw response: ${text.slice(0, 200)}`

No date-fns — use raw Date math for month string as shown above.
