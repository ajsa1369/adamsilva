# Environment Variables

Tags: #config #env

All vars live in `.env.local` locally and Vercel dashboard for production.
`.env.local` is gitignored — never committed.

## Supabase
| Var | Scope |
|-----|-------|
| `NEXT_PUBLIC_SUPABASE_URL` | Public |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Public |
| `SUPABASE_SERVICE_ROLE_KEY` | Server only ⚠️ |

## Strapi
| Var | Value |
|-----|-------|
| `NEXT_PUBLIC_STRAPI_URL` | https://cms.adamsilvaconsulting.com |
| `STRAPI_API_TOKEN` | Full access token from Strapi admin |

## Site
| Var | Value |
|-----|-------|
| `NEXT_PUBLIC_SITE_URL` | https://www.adamsilvaconsulting.com |

## Email (SMTP)

| `SMTP_PORT` | 465 |
| `SMTP_USER` | admin@adamsilvaconsulting.com |
| `SMTP_PASS` | (in .env.local) |
